## Español

¿No sientes que a veces es imposible concentrarte?
Con **Pomodoro Timer**, tus sesiones de estudio o trabajo pueden volverse mucho más efectivas.

Esta aplicación está basada en la técnica **Pomodoro**, un método de gestión del tiempo que divide el trabajo en intervalos enfocados (normalmente 25 minutos) seguidos por pequeños descansos. La idea es simple: trabajar con máxima concentración durante un periodo corto y repetir el proceso varias veces.

Este proyecto busca ofrecer una experiencia **intuitiva, sencilla y clara**, para que puedas concentrarte en lo importante sin distracciones.

### Características

* Temporizador de **25 minutos** listo para empezar.
* Botones para **Start, Stop y Reset**.
* Contador de **sesiones completadas**.
* Ajuste rápido del tiempo con botones **+5 y -5 minutos**.
* Interfaz simple y fácil de usar.

### Objetivo del proyecto

Este proyecto fue desarrollado como práctica de desarrollo web, enfocándose en:

* Manejo del estado de una aplicación.
* Manipulación del tiempo y eventos.
* Creación de interfaces simples pero funcionales.
* Construcción de aplicaciones interactivas para mejorar la productividad.

### Cómo usar la aplicación

1. Ajusta el tiempo si lo deseas con **+5 o -5 minutos**.
2. Presiona **Start** para comenzar la sesión.
3. Trabaja con máxima concentración hasta que el temporizador termine.
4. Repite el proceso para completar varias sesiones de enfoque.

### Tecnologías utilizadas

* HTML
* CSS
* JavaScript

## English
You can try the app here: 
👉 https://pomodorotimer-eta.vercel.app/

Do you ever feel like you simply can't concentrate?

With **Pomodoro Timer**, your study and work sessions can become far more effective.

This application is based on the **Pomodoro Technique**, a time-management method that breaks work into focused intervals (usually 25 minutes) followed by short breaks. The idea is simple: work with full focus for a short period of time and repeat the cycle multiple times.

This project aims to provide an **intuitive and simple experience**, allowing users to focus on productivity without unnecessary distractions.

### Features

* **25-minute timer** ready to start.
* **Start, Stop, and Reset** controls.
* **Session counter** to track progress.
* Quick time adjustment with **+5 and -5 minutes**.
* Clean and easy-to-use interface.

### Project goal

This project was created as a web development practice project, focusing on:

* Application state management
* Handling timers and events
* Building simple but functional interfaces
* Creating interactive tools to improve productivity

### How to use

1. Adjust the timer using **+5 or -5 minutes** if needed.
2. Press **Start** to begin a focus session.
3. Work with full concentration until the timer finishes.
4. Repeat the process to complete multiple productive sessions.

### Technologies used

* HTML
* CSS
* JavaScript
